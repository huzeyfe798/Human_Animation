# Human_Animation


press a : Hit a ball
press y : Walk
press q : Animation Between the current position and the animation.txt 
press e : Shake hand
press r : Reset 

Install 

Ubuntu:
sudo apt-get install freeglut3-dev

How to run it

g++ 14290068and14290044.cpp -lglt -lGLU -lGL -lm

./a.out
