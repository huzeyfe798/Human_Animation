# Human_Animation
Used library : Open GL <GL/glut.h>

press a : Hit a ball
press y : Walk
press q : Animation Between the current position and the animation.txt 
press e : Shake hand
press r : Reset 3

Install 

Ubuntu:
sudo apt-get install freeglut3-dev

How to run it

g++ humananimation.cpp -lglt -lGLU -lGL -lm

./a.out
